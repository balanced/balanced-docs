Categories
==========

For a FAQ item to show up in the help section on the [website](https://www.balancedpayments.com/help), there must be a category attribute on the FAQ item.

Current list of categories
--------------------------

+ payouts
+ signup
+ chargebacks
+ processing
+ fees
+ security
+ fraud
+ data
+ legal
+ marketing
+ end_user
